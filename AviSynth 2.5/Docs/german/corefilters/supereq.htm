SuperEQ

SuperEQ(string filename)

Shibatch Super Equalizer ist ein grafisches und parametrisches Equalizer Plugin für WinAMP. Dieses Plugin nutzt den 16383zigsten FIR Filter mit dem FFT Algorithmus. Dessen Equalisierung ist sehr genau. Die Einstellungen können für jeden Kanal einzeln getätigt werden. SuperEQ ist ursprünglich von Naoki Shibata geschrieben worden.

SuperEQ benötigt eine foobar2k Equalizer Einstellungsdatei. Den Equalizer findet man in foobar's DSPManager und die Einstellungen können hier gewählt und dann abgespeichert werden. Einige Standard-Einstellungen kann man hier herunterladen.

Dieses Plugin ist für Prozessoren optimiert worden, die einen internen Cache von 128k bytes (16383*2*sizeof(float) = 128k) oder größer haben. D.h. dieses Plugin funktioniert nicht sehr effizient mit der AMD K6 Serie, wohl aber mit AMD Athlon`s.

Parameter:
filenameHier gibt den Dateiname der foobar2k Equalizer Einstellungsdatei an.

Beispiele:
SuperEQ("C:\Equalizer\Loudless.feq")Verwendet den "Lautlos" Filter von obigen Standard-Einstellungen
