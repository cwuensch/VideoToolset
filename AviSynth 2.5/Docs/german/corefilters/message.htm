MessageClip

MessageClip(string "message", int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

Es wird ein Clip mit einer Textzeile generiert. Wird zum Beispiel intern für die Fehlermeldungen verwendet. Eine Arial-Schriftart wird verwendet, eine Schriftgröße zwischen 9 und 24 Punkten wird automatisch gewählt, damit die Schrift in das Video passt. Die Breite und Höhe kann mit width und height angegeben werden. Bei shrink=true wird die Größe vermindert, damit der Text genau hineinpasst.
