ResampleAudio

ResampleAudio(clip, int "new_sample_rate")

Dieser Filter führt eine qualitativ hochwertige Umwandlung der Samplingrate durch.
Eine Umwandlung wird übersprungen, falls die Samplingrate schon der aktuellen Rate entspricht.

Beispiel:

Resamples den Ton auf 48 kHz:
AviSource("C:\Dateiname.wav")
ResampleAudio(48000)
