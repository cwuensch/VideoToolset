FreezeFrame

FreezeFrame(clip, int "first_frame", int "last_frame", int "source_frame")

Dieser Filter ersetzt alle Frames zwischen first_frame und last_frame mit einer Kopie von source_frame. Der Ton wird nicht verändert. Dies kann zum Ausbessern von Fehlern verwendet werden, bei denen man für defekte Frames ähnliche, fehlerfreie Frames verwenden kann.
