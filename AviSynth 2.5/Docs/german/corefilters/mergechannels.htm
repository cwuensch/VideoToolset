MergeChannels

MergeChannels(clip1 , clip2 [, clip3, ...])

Seit AviSynth v2.5 ersetzt MergeChannels MonoToStereo und kann genutzt werden, um die Tonkanäle zweier oder mehrerer Clips zu mischen.

Beispiele:

# Konvertiert eine unkomprimierte WAV Datei in ein 44.1 kHz Stereo Signal:
video = AviSource("C:\Dateiname.avi")
audio = WavSource("C:\Dateiname.avi")
l_ch = GetChannel(audio, 1)
r_ch = GetChannel(audio, 2)
stereo = MergeChannels(l_ch, r_ch).ResampleAudio(44100)
return AudioDub(video, stereo)

# Alternativ:
video = AviSource("C:\Dateiname.avi")
audio = WavSource("C:\Dateiname.avi")
stereo = GetChannel(audio, 1, 2).ResampleAudio(44100)
return AudioDub(video, stereo)
