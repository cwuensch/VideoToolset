Compare

Compare(clip_filtered, clip_original, string "channels", string "logfile", bool "show_graph")

Compare ermöglicht einen Vergleich zwischen einem gefilterten Clip und dem Original. Dazu wird Text (Mean Absolute Difference, Mean Difference, PSNR und Overall PSNR) und eine optional auch eine Grafik (Marc`s PSNR Graph) im Bild ausgegeben.

Parameter:
clip_filteredHier gibt man den gefilterten Clip an.
clip_orignalGibt den Original-Clip an.
channelsMit diesem optionalen Parameter kann man den Vergleich auf bestimmte Farb- und/oder Helligkeitskanäle beschränken. Je nach Farbraum der Quellclips ist eine Kombination aus R,G,B[,A] oder Y,U,V erlaubt. Ein leerer String wird als YUV oder RGB interpretiert.
logfileWenn man bei diesem optionalen Parameter den Dateinamen einer Logdatei (in Anführungsstrichen!) angibt, erfolgt die Ausgabe des Textes nur in dieser Datei und nicht mehr auf dem Bildschirm. Dies ist viel schneller und besonders nützlich beim Vergleich von längeren Clips. Hinweis: "Overall PSNR" ist noch nicht in der Logdatei enthalten.
show_graphMit diesem optionalen Parameter kann man bei true Marc`s PSNR Graph anzeigen lassen.

Beispiele:
Compare(clip1,clip2)Zeigt die Unterschiede am Bildschirm an
Compare(clip1,clip2,"compare.log")Erzeugt nur eine Logdatei
Compare(clip1,clip2,"UV")Vergleicht nur das Farbsignal von zwei YUY2 Clips
