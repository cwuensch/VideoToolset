SelectRangeEvery

SelectRangeEvery(clip, int "period", int "range")

Dieser Filter ist seit AviSynth v2.5 verfügbar. Er wählt einen Bereich an Frames mit einer definierten Periode aus einem Quellclip.

Beispiel:

# Wählt die Frames 0 bis 13, 280 bis 293, 560 bis 573 usw. aus:
SelectRangeEvery(clip, 280, 14)
