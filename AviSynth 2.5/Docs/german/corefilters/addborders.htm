AddBorders

AddBorders(clip, int "left", int "top" [, int "right", int "bottom"] [, int "color"])

AddBorders fügt schwarze Ränder mit der angegebenen Anzahl von Pixeln zum Bild dazu.

Siehe Farbraumkonvertierungs Filter für die Besonderheiten der einzelnen Farbräume.

Der color Parameter ist optional (verfügbar seit v2.07), Standard=0 <black>, und wird als RGB Wert angegeben unabhängig davon, ob der Farbraum RGB oder YUV basierend ist.

Achtung: viele Kompressionsalgorithmen haben Schwierigkeiten mit vollfarbigen Rändern, wenn die Breite der Ränder nicht ein Vielfaches der Blockgröße ist (16 Pixel bei MPEG).

Dieser Filter kann zusammen mit Crop verwendet werden, um ein Bild zu verschieben, ohne die Bildgröße zu verändern:

Beispiel:

# rückt ein 352x240 Bild um 2 Pixel nach rechts:
Crop(0,0,350,240).AddBorders(2,0,0,0)
