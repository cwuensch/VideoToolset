Import

Import(string filename)

Import importiert den Inhalt eines anderen AviSynth Skriptes.
Dies ist sehr nützlich, wenn man komplexe Skript-Funktionen, globale Variablen und vorbearbeitete Clips wiederverwenden will.

Beispiele:
Import(AddAudio.avs).AddAudio()Importiert eine benutzerdefinierte Funktion und führt sie aus
