SelectEvery

SelectEvery(clip, int step_size, int offset1 [, int offset2,...])

Mit SelectEvery kann man aus einem Quellclip bestimmte Frames auswählen, die bestimmten Regeln folgen.
SelectEvery ist die Verallgemeinerung von Filter wie SelectEven und Pulldown.

Parameter
step_size Hiermit wird als Ganzzahl angegeben, der wievielte Frame immer ausgewählt wird.
offset1 Hiermit gibt man als Ganzzahl einen Versatz, um nicht gleich mit dem ersten Frame anzufangen. Es können beliebig viele weitere ''offsets'' angegeben werden, die dann als weitere Startpunkte der Auswahlfolge fungieren.

Beispiele
Dasgleiche wie SelectEven():
SelectEvery(clip,2,0)
Dasgleiche wie SelectOdd():
SelectEvery(clip,2,1)
Verwendet Frames 3,6,7,13,16,17,... vom Quellclip:
SelectEvery(clip,10,3,6,7)
Verwendet Frames 0,9,18,... vom Quellclip:
SelectEvery(clip,9,0)
Auf einen 24 fps progressiven Quellclip 3:2 Pulldown anwenden, was einen 30 fps interlaced Clip ergibt:
AssumeFrameBased()
SeparateFields()
SelectEvery(8,0,1,2,3,2,5,4,7,6,7)
Weave()
