Limiter

Limiter(clip, int "min_luma", int "max_luma", int "min_chroma", int "max_chroma")

Dieser Filter ist seit AviSynth v2.5 verfügbar. Die Norm CCIR-601 definiert Bereiche an gültigen Pixelwerten für die Darstellung auf einem Fernseher. Diese Bereiche sind 16-236 für die Helligkeitskomponente und 16-240 für die Farbinformationen.

Pixelwerte, die außerhalb dieses Bereiches liegen, verursachen Probleme mit manchen Fernsehern. Dieser Filter ist der beste Weg, um diese ungültigen Pixelwert vor dem Kodieren zu entfernen, wenn man sich das Ergebnis auf dem Fernseher betrachten will. Standardmäßig setzt dieser Filter Pixelwerte unter 16 auf 16 und über 236 (oder 240) auf 236 (oder 240).
