Blur / Sharpen

Blur(clip, float "amount")
Blur(clip, float "amountH", float "amountV")
Sharpen(clip, float "amount")
Sharpen(clip, float "amountH", float "amountV")

Diese zwei Filter sind einfache 3x3-Kern Unschärfe- bzw. Schärfe-Filter. Der größte erlaubte Wert ist 1.58, das entspricht einer gleichmäßigen 3x3-Pixel Unschärfe. Ein Wert von 1.0 entspricht einem (1/4,1/2,1/4) Kern. Für gaussche Unschärfen mit größerem Radius können mehrere Befehle Blur(1.0) hintereinander gereiht werden.

Negative Argumente von Blur schärfen das Bild, da Sharpen( x ) nur eine andere Schreibweise ist für Blur( -x ). Der kleinste erlaubte Wert für Blur ist -1.0, der größte für Sharpen ist 1.0.

Man kann auch bei den beiden Filtern zwei Argumente angeben, um horizontal und vertikal unabhängig voneinander zu filtern. So kann man mit Blur(0,1) nur vertikal filtern, um z.B. interlaced Zeilen miteinander durch Überblendung zu verbinden.
